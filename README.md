# Actividad Clase 4 - Git y GitHub

## Resumen
En esta actividad se utilizó Git y GitHub para gestionar versiones de un proyecto. Se creó un repositorio remoto, se vinculó con el proyecto local mediante SSH y se publicaron los cambios utilizando los comandos básicos de Git (`status`, `add`, `commit` y `push`).

## Herramientas
- Git
- GitHub
- Git Bash
- SSH
